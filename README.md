# VYNTRA - Classic BMW Sales in Botswana

A modern, elegant website for selling classic BMW automobiles in Botswana.

## Pages

- **Home** (index.html) - Landing page with hero section and features
- **Shop** (shop.html) - Browse classic BMW collection with prices in Botswana Pula
- **About** (about.html) - Information about VYNTRA
- **Contact** (contact.html) - Contact form and location details
- **Login** (login.html) - Customer login page

## Features

✨ Modern and elegant design
✨ Responsive layout for all devices
✨ Classic BMW models from Gaborone, Francistown, Maun, and Bobonong
✨ Prices displayed in Botswana Pula (P)
✨ Beautiful product cards with image support
✨ Smooth hover animations
✨ Professional navigation and branding
✨ Call-to-action buttons and contact banner
✨ Clean, semantic HTML structure
✨ Minimal CSS with no JavaScript
