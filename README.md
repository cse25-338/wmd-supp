# VYNTRA - Classic BMW Sales

A simple and elegant website for selling classic BMW automobiles in Botswana.

## Pages

- **Home** (index.html) - Landing page with hero section
- **Shop** (shop.html) - Browse classic BMW collection with prices in Botswana Pula
- **About** (about.html) - Information about VYNTRA
- **Contact** (contact.html) - Contact form and location details
- **Login** (login.html) - Customer login page

## Features

- Simple and elegant design
- Responsive layout for all devices
- Classic BMW models from Gaborone, Francistown, Maun, and Bobonong
- Prices displayed in Botswana Pula (P)
- Minimal CSS and no JavaScript
- Clean HTML structure
